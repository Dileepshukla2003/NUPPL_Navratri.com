Navratri 2025 - Static Website
Files:
- index.html
- styles.css
- script.js

How to use:
1. Unzip navratri_website.zip
2. Open index.html in your browser. No server or database required.

This package includes an inline SVG banner labeled "Navratri 2025" and stylized SVG goddess icons for each day.
Feel free to replace the SVG art in index.html with real goddess photos (put them in /assets and update <img> tags).
